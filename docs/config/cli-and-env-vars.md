# CLI & Env Vars

## Config Path

| Environment Variable | CLI Flag           | Default      |
|----------------------|--------------------|--------------|
| `INFRARED_CONFIG`    | `--config` or `-c` | `config.yml` |

## Working Directory

| Environment Variable   | CLI Flag                | Default |
|------------------------|-------------------------|---------|
| `INFRARED_WORKING_DIR` | `--working-dir` or `-w` | `.`     |

## Proxies Path

| Environment Variable   | CLI Flag                | Default     |
|------------------------|-------------------------|-------------|
| `INFRARED_PROXIES_DIR` | `--proxies-dir` or `-p` | `./proxies` |
