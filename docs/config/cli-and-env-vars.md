# CLI & Env Vars

## Config Path

| Environment Variable | CLI Flag         | Default      |
|----------------------|------------------|--------------|
| `INFRARED_CONFIG`    | `--config`, `-c` | `config.yml` |

## Working Directory

| Environment Variable   | CLI Flag              | Default |
|------------------------|-----------------------|---------|
| `INFRARED_WORKING_DIR` | `--working-dir`, `-w` | `.`     |

## Proxies Path

| Environment Variable   | CLI Flag              | Default     |
|------------------------|-----------------------|-------------|
| `INFRARED_PROXIES_DIR` | `--proxies-dir`, `-p` | `./proxies` |

## Log Level

| Environment Variable | CLI Flag            | Default |
|----------------------|---------------------|---------|
| `INFRARED_LOG_LEVEL` | `--log-level`, `-l` | `info`  |
